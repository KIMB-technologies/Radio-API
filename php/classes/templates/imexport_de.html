Der Import und Export ist für diese Installation deaktiviert! Bitte setzen Sie in der Konfiguration den Wert CONF_IM_EXPORT_TOKEN auf einen möglichst zufälligen Token mit mindestens 15 Zeichen aus der Menge A-Z, a-z und 0-9. Damit wird der Import und Export aktiviert und ist hier mittels des konfigurierten Tokens möglich.

Nutzen Sie z.B. den Token %%EXAMPLETOKEN%%.

Ein Export ist eine Export-Datei (*.json). Es werden immer alle bekannten Radios dieser Installation von Radio-API exportiert, d..h, für jedes Radio die Liste der Radiosender, Podcasts, zuletzt per RadioBrowser gehörten Station und bereist gehörten Podcast-Episoden. Auch wird die Liste zur Zuordnung von GUI-Codes zu Radios exportiert.

Sie können den Export auch automatisieren und so z.B. regelmäßige Backups zu erstellen. Der Aufruf von %%SERVERURL%%gui/im-export.php?task=export&token=<token> gibt eine Export-Datei zurück.

Bestätigung Wollen Sie wirklich alles überschreiben?
Bitte wählen Sie eine Export-Datei (*.json) für den Import und geben Sie den Token zur Authentifizierung an. Sie können auswählen, ob Sie die Inhalte aus dem Export beim Import hinzufügen, nur ein einzelnes Radio überschreiben oder alles ersetzen wollen.
Hinzufügen
Alle Radios und Elemente aus dem Export werden dieser Installation von Radio-API hinzugefügt. Es werden also weitere Radios für diesen Server eingerichtet. Stellen Sie sicher, dass die Radios (deren Macs) sich vorher noch nicht mit diesem Server verbunden haben.
Einzelnes Radio überschreiben
Überschreiben Sie die Inhalte eines einzelnen Radios aus dem Export mit einem einzelnen Radio in dieser Installation von Radio-API. Geben Sie dazu den GUI-Code des Radios im Export an (was sie importieren wollen) und den GUI-Codes des Radios im System/ Installation an (in das sie importieren wollen). Sie überschreiben damit alle Daten des einen Radios in dieser Installation von Radio-API.
Alles ersetzen
Ersetzen Sie die Inhalte dieser Installation von Radio-API vollständig mit den Inhalten des Exports. Es werden alle Daten überschrieben!
Starten Sie den Import schließlich über den Button Import starten, fehlerhaften Felder werden rot markiert.
